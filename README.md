## Caracteristicas

- **Dashboard Interactivo**: Visualizacion de metricas y distribucion de datos.
- **Prediccion en Tiempo Real**: Formulario para predecir riesgo de nuevos estudiantes.
- **Visualizacion de Clusters**: Graficos interactivos para analizar perfiles de aprendizaje.
- **Documentacion Academica**: Tesis completa integrada en el sistema.
- **Responsive Design**: Adaptado para dispositivos moviles y de escritorio.
- **Accesibilidad**: Soporte para navegacion por teclado y lectores de pantalla.

## Tecnologias Utilizadas

- **HTML5**: Estructura semantica y accesible.
- **CSS3**: Estilos modernos con variables CSS, Grid y Flexbox.
- **JavaScript ES6+**: Logica de aplicacion y manipulacion del DOM.
- **Chart.js 4.4.1**: Visualizacion de datos interactivos.
- **Tailwind CSS**: Framework de utilidades CSS (via CDN).
- **Google Fonts**: Tipografias Space Grotesk y DM Sans.
